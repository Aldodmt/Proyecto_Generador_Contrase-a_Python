# Generador de contraseña en Python 

La funcion de este proyecto es practicar mis capacidad y en este caso fue un generador de contraseña
de cadenas minusculas, mayusculas, numeros y simbolos

# Funcionalidades
```
Permite generar contraseñas con la longitud que el usuario defina
Permite generar:
-Mayuscula
-Minuscula
-Numericos
-Simbolos
```
